SIMON SAYS... MAKE A GAME!

    Technologies Used : html, css, javascript, jquery, github, gh pages, amazon aws

My first project for General Assembly's Web Developer Immersive was a front-end language coding challenge. Assigned because it would assist with becoming familiar with HTML, CSS, Javascript, and jQuery, I was tasked with creating a game from scratch from a short list. I chose Simon as I was very familiar with the game. If wasn’t until after I made my choice that I realize I chose one of the more intensive games to build.

I assumed since I was strong in html and css and familiar with javascript plugins, this would be a breeze. It definitely was not. There were many times during the build my entire functionality would become useless. Being new to vanilla javascript syntax, much of my code would not run after I would add a new function. I did more debugging than coding but it was a great learning experience. After many issues, I took a step back, reviewed my learning materials and also found some online references and industry leading book, and was successfully able to build a functional game. I used javascript and jquery documentation and libraries to figure out the building of all the functionality.
